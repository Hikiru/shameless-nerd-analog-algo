# Shameless Nerd Analog Algorithm
Website inspired by Reid from Shameless Nerd's "analog algorithm" google forms, used to gather community-based recommendations rather than ones by a computer.

# Current overarching goals:
- Various questions for movies, games, or other media people enjoy and want others to see
- Compiled list of top choices every week
- Chat rooms for discussion of current week's top choices
